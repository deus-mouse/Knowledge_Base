pwd
Вывести текущую (рабочую) директорию.

date
Вывести текущую дату и время системы.

w
Данная команда показывает, кто залогинен в системе.
Помимо этого также на экран выводится uptime и LA (load average).

ls
Вывести содержимое директории. Если не передать путь,
то выведется содержимое текущей директории.


cd
Изменить текущую директорию.


mkdir
Создать директорию.
mkdir test2/subtest

rm
Удалить файл.
$ rm <filename>
$ rm -rf <filename>

cp
Копировать файл или директорию.
$ cp temp temp_clone

mv
Переместить или переименовать файл или директорию.
mv test test_renamed

cat
Вывести содержимое файла (или файлов).
$ cat temp

tail
Вывести n последних строк или байт файла.
$ tail -1 temp

less
Иногда текстовый файл слишком большой, и неудобно выводить его командой cat.
Тогда можно открыть его с помощью команды less: файл будет выводиться по частям,
доступна навигация по этим частям, поиск и прочий простой функционал.
$ less temp

ps
Вывести список процессов.
$ ps

kill
Послать сигнал процессу. По умолчанию посылается сигнал SIGTERM, который завершает процесс.
$ kill 1033727

ping
Послать хосту ICMP пакет ECHO_REQUEST.
$ ping google.com

ssh
OpenSSH SSH клиент, позволяет подключаться к удаленному хосту.
$ ssh user@11.11.22.22

scp
Копировать файлы между хостами (для этого используется ssh).
$ pwd

rsync
Также для синхронизации директорий между хостами можно использовать rsync
(-a — archive mode, позволяет скопировать полностью всё содержимое директории «как есть»,
-v — вывод на консоль дополнительной информации):
$ ls Downloads/user

echo
Вывести на экран строку текста.
$ echo "Hello"

telnet
Клиент для протокола TELNET. Используется для коммуникации с другим хостом.
$ telnet example.com 80
